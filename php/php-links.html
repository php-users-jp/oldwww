PHPがサポートするデータベース

PostgreSQL 高機能でかつ無償で使用可能なデータベースです。 日本語処理にも対応しており、フリーUNIXユーザーに大変人気があります。
PostgreSQL (オブジェクト指向RDBMS)に関する日本語MLのページ
MySQL 非商用用途に関して無償で使用可能なデータベースです。 高速性が売りです。 日本語の処理にも対応しており、特に海外では人気があるようです。
詳細は、 「MySQLのページ」を参照下さい。
(有志の方により日本語マニュアル等も整備されています。)
Mini SQL
(mSQL)		 高速で軽いRDBMSです。シェアウエアですが、学術関係等は無償で使用できます。
http://www.hughes.com.au/ をご覧下さい。
Oracle Oracle は、実績のあるデータベースシステムであり多くの実用システムで使用されています。 Oracle 8 は、Linux 版もリリースされています。
http://technet.oracle.com/ をご覧下さい。

下岡＠アクシスソフトウェアさんのページ： http://www.pat.hi-ho.ne.jp/dimension/php/ にて Oracle+PHP の接続に関する情報が公開されています。

Sybase 歴史のあるRDBMS製品の一つであり、SQL Server は、高性能・高信頼性を有する製品として ミッションクリティカルな分野で使用されています。 Linux 版 Adaptive Server Enterprise の開発ライセンスは現在無償となっており、 以下のURLにおいて入手可能です。
http://linux.sybase.co.jp/ をご覧下さい。
ODBC MS-Windows環境におけるデータベースインターフェースの標準です。 MS SQL Server を初めとする多くのデータベースがODBCに対応しています。
UNIX環境からMS-Windows上で動作するODBCインターフェースを有するデータベースサーバー にアクセスするためには、ODBCインターフェース iODBC をインストールし、PHPに組み込む必要があります。 iODBC は、http://www.iodbc.org/ にて取得できます。
