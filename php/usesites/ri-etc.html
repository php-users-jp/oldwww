Ground　SunLight ( http://www.y2sunlight.com/ground/ )
このサイトのテーマは「Windowsで作るWebプログラミングの開発環境(PerlとPHP)」です。 現在は「無料で揃える開発環境」を連載中です。 岩田善洋 様
レンタルサーバにアップロードする前に、手元のWindows機でプログラムを テストしたい人にはお勧めの内容です。また、御用とお急ぎ手ない方は、 壮年プログラマの日々の思いを朝礼風に綴った「プログラマの朝礼」も ご覧になって下さい。
テスト項目運用システム ( http://iserlone.minidns.net/programs/tisystem/ )
ソフトウェア開発におけるテスト項目の作成・実施、障害票の発行・回答、 進捗の管理を行うＷｅｂ−ＤＢを公開しています。 ＰＨＰ３（又は４） + PHPlib + PostgreSQL + Apache で動作します。 憂鬱なテスト工程でのテスター・プログラマ・マネージャの負担を軽減で きればと思って作りました。 井田 様
Ｗｅｂページでご試用いただけます。 カンパウェアですが、カンパなしでも制限なく使用できます。
スクリプト言語ソースツアー ( http://www.tamacom.com/langtour-j.html )
本サイトでは、PHPの処理系とphpGroupWareのソースコードをハイパーテキスト化 して公開致しております。 山口@多摩通信社 様
ハイパーテキスト作成のためのツールもGNU GPLの元で配布しております。 大規模PHPサイトの解析やレビューにご活用ください。
PHPFAQ-JP ( http://phpfaq.eai.jp/ )

Webグループウェア La!cooda WIZ (らくーだウィズ) ( http://wiz.syscon.co.jp/ )
ユーザ数無制限で無償かつオープンソースで提供している Webグループウェアで す。 スケジュール帳、回覧板、行先ボード、伝言メモ、ワークフロー、ファイ ル共有、 掲示板など11種類の基本機能があります。 wiz@syscon.co.jp 様

ACHROMAT.NET ( http://achromat.net/ )
PHPをはじめいくつかのプログラミング言語や、 Linux等に関する事の入門段階を解説していきます。 achromat.net 様
サイト自体、９０％はPHPとMySQLで管理 しています。みなさんのリクエスト等もどんどん受けていきたいと思っています。 現在、雑多になってはいますが、それぞれもっとしっかりまとめていく予定です。
bookmark+x ( http://chmodx.dyndns.org/aaf/bookmarkx/bookmarkx.html )
PHP+PostgreSQLで動作するブックマーク管理アプリケーションです(フリーソフトウェア)。 複数のユーザで使ったり、ブックマークを公開する機能もあります。 こもりや＠AAFぱ研 様
PHP,PostgreSQLともに初めて使ったのですが、とても快適ですね。 開発途上で不十分な点があるとは思いますがよろしくお願いします。
Mac OS Xにオープンソースソフトをインストール ( http://www.yo.rim.or.jp/~t_okuno/aphpgsql.html )
