PHP3 日本語マニュアル作成プロジェクト

プロジェクトについて

PHP の新版,PHP3 の日本語マニュアルをオリジナルの英文マニュアルを 和訳することにより作成するプロジェクトです。
翻訳作業に協力していただける方を募集しています。

プロジェクトへの参加方法

sgk さんが日本語マニュアル作成プロジェクト用の ページを作成して下さいました。 プロジェクトは、メーリングリストとCVSによるシステムで 作業を行っています。 こちらでプロジェクトへの 参加方法、作業の進行状況等が参照できます。 また、ご質問がある方は、広川まで メールを下さい。

配布条件・権利などについて

作成した翻訳ドキュメントについての権利関係については、 以下の方針にそって定義したいと思います。
    　　
  • 無償で公開する。 　　
  • 配布には特に条件を設けない。 　　
  • 改変および二次使用に制限を設けない。 　　
  • 翻訳著作にかかる著作者は明記するが、人格権以外の権利は主張しない。
最終的には、PHP3 本家のアーカイブの方にコントリビュートすることを 目的としたいと思います。

翻訳の進め方（案）

PHP3 からは旧版と異なり、ドキュメントは SGML 形式で記述されています。 HTML他の形式への変換は、JADE などの SGML パーサを使うようになって いるようです。

本翻訳プロジェクトでも、本家との互換性を維持するために、SGML ファイル ベースでの翻訳を行い、JADE 等のツールにより HTML 形式への変換を行う こととします。

JADE、DTD ファイルは、オリジナルのものを使用し、日本語化することは 現在のところ考えていません。

日本語文字コードは、今後、Win32 環境で PHP3 の使用が広まるであろうことから、 Shift-JIS を使用したいと思います。

翻訳のオリジナルは、PHP3 の Release 版とします。

翻訳状況(1999/11/11現在)

PHP3.0.10 以降、SGML ファイルが配布ファイルに含まれなくなったため、 開発用CVSツリーから直接入手して翻訳を行っています。 このため、特定のバージョンに対応するのではなく、 基本的に全ての機能(開発中の機能に関する記述も含む)を含むようになっています。

JADE により HTML ファイルに変換したファイルを公開します。

翻訳等に誤り等ございましたら、広川または日本語訳作成mlに て指摘ください。
