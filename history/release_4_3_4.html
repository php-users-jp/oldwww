PHP 4.3.4 Release Announcement

長期に渡るQAプロセス(訳注：Quality Assurance)を経て、 PHP 4.3.4がついにリリースされました。
このリリースは変更点としては中規模のメンテナンスリリースであり、 かなりの数のバグが修正されています。全てのユーザに対して4.3.4への アップグレードが推奨されます。

Bugfix Release

PHP 4.3.4では重要な修正、機能追加、改善が行われています。 その一部を以下にあげます：

  • disk_total_space()とdisk_free_space()のFreeBSDでの挙動を修正
  • FastCGIが特定のIPアドレスにバインドできない ("-b IP:port"が正しく動作しない)問題を修正
  • win32でのmail()関数のバグ修正
  • 関数内での多くのクラッシュを修正
  • MacOSX 10.3 Pantherでコンパイルが失敗する問題の修正
  • 60以上のバグフィクス

PHP 4.3.4における変更点の一覧についてはNEWSファイルを参照してください。
