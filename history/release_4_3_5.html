PHP 4.3.5 Release Announcement

PHP開発チームは自信を持ってPHP 4.3.5のリリースをお知らせします。 PHP 4.3.5は基本的にバグフィックスリリースであり、新機能の追加はありません。 PHP 4.3.5はこれまでで最も安定したリリースであり、 全てのユーザは可能な限りこのバージョンへのアップグレードが推奨されます。
Bugfix Release

PHP 4.3.5では重要な修正、機能追加、改善が行われています。 その一部を以下にあげます：

  • Apacheの他のバーチャルホストのini設定を読み込めてしまう問題を修正
  • fgetcsv()のクラッシュを修正し、バイナリセーフに変更
  • GCC 3.0の初期バージョンでのコンパイルに対応
  • ソケットに対してfeof()が正しく動作しない問題を修正
  • get_browser()関数のパターンマッチアルゴリズムを改善
  • Win32版open_basedirのパス名解決を修正
  • safe_modeがonの場合に存在しないディレクトリに対する操作を 行った場合の不正なエラーを修正
  • Win32版にバンドルされたOpenSSLのdllバージョンを0.9.7cに アップグレード
  • Win32版にバンドルされたPostgreSQLライブラリのバージョンを 7.4にアップグレード
  • バンドルされたPCREライブラリのバージョンを4.5にアップグレード
  • バンドルされたGDライブラリのバージョンを2.0.17にアップグレード
  • 64bitシステムに対応した数々の修正
  • その他140以上のバグフィクス

PHP 4.3.5における変更点の一覧についてはNEWSファイルを参照してください。
