PHP 4.1.1 Release Announcement

PHP 4.1.0には幾つかのバグがあったためPHP 4.1.1をリリースすることに しました。修正されたバグは重大なものではなく、少し困ってしまう、 という程度のものです。
既にPHP 4.1.0へアップグレードした方は、以下に示すバグが問題と なっているのでない限りPHP 4.1.1にはアップグレードしないことを お勧めします。

このリリースでは、新しい機能の追加やセキュリティの更新はありません。

修正項目一覧:

  • .NET/IIS 6との互換性の問題の解決 - これにより他のバージョンのIISでの
    動作の安定性も改善される (Zeev)
  • スレッドを使用するプラットフォームで、スクリプトをシャットダウン するときにクラッシュしたりエラーが発生するバグの修正 (Zeev)
  • xslt拡張モジュールでのクラッシュバグの修正 (Markus, Derick)
  • dbaseでlongより大きい値の整数を正しく返さない問題の修正 (Vlad)
  • domxml拡張モジュールのバグとメモリリークの修正 (Markus)
  • タイムゾーンが世界標準時になっているとgmmktime()のかえす値が 1時間ずれる問題の修正 - bug #9878. bfoddy@mediaone.netのパッチに拠る (jmoore)
  • Windowsでのgmdate()のタイムゾーンの扱いに関するバグの修正 - bug #13885 (jmoore)
  • mcrypt拡張モジュールのクラッシュバグの修正 (Derick)
  • mcrypt拡張モジュールを、libmcrypt 2.2を使用してコンパイルする ように再度変更 (Sterling)
  • is_file()系の関数に存在しないファイル名が与えられた場合に、 エラーではないのにエラーを出力することがある問題の修正 (Sterling)
  • strtotime()が GMT +0100とGMT -0100を正しく認識していない バグの修正(Derick)
